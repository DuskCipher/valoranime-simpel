# 🚀 Valoranime - Platform Streaming Anime, Donghua & Baca Komik/Novel (Simpel & Database-Free)

Selamat datang di *Source Code* **Valoranime-Simpel**! Ini adalah platform modern berbasis **Next.js** untuk menonton Streaming Anime, Donghua, dan membaca Komik (Webtoon) serta Novel secara gratis, tanpa iklan, dan **100% Bebas Database (Zero Database Dependency)**.

---

## ✨ Fitur Utama
- 📺 **Streaming Anime & Donghua**: Nonton episode terbaru lengkap dengan jadwal rilis, daftar genre, dan pencarian.
- 📖 **Baca Komik & Novel**: Pembaca chapter komik/webtoon dan novel teks yang responsif.
- 🔍 **Pencarian Global**: Cari anime, donghua, komik, atau novel secara instan.
- 📚 **Perpustakaan Saya (LocalStorage)**: Riwayat tontonan/bacaan dan bookmark otomatis tersimpan di browser tanpa memerlukan server database.
- ⚡ **Super Cepat & Bebas Database**: Tidak membutuhkan Supabase, PostgreSQL, atau backend database apapun.

---

## 🛠️ Persiapan Awal (Requirements)
1. **Node.js** (versi 18 atau lebih baru)
2. **npm** (termasuk saat menginstall Node.js)
3. Akun **Vercel** atau platform hosting sejenis (untuk deploy online)

---

## 💻 Menjalankan Web di Komputer (Lokal)

1. **Clone repository ini**:
   ```bash
   git clone https://github.com/DuskCipher/valoranime-simpel.git
   cd valoranime-simpel
   ```

2. **Install Dependensi**:
   ```bash
   npm install
   ```

3. **Jalankan server lokal (Development Mode)**:
   ```bash
   npm run dev
   ```

4. Buka browser dan pergi ke `http://localhost:3000`.

---

## 🚀 Deploy ke Vercel

1. Push repository ini ke GitHub Anda.
2. Login ke [Vercel](https://vercel.com/) dan buat project baru.
3. Import repository GitHub Anda.
4. Klik **Deploy** *(Tidak memerlukan Environment Variables database Supabase)*.
